# Product Overview

## Multilingual Mandi

A progressive web application (PWA) designed to modernize India's traditional mandi (wholesale market) ecosystem by breaking language barriers and improving price transparency between vendors and buyers.

### Core Purpose
- Enable multilingual communication across 22+ Indian languages
- Provide real-time market price discovery and transparency
- Facilitate fair negotiations with AI-powered assistance
- Connect farmers, wholesalers, retailers, and urban buyers
- Work offline-first for areas with poor connectivity

### Key User Types
- **Vendors**: Small-scale sellers, farmers, wholesalers
- **Buyers**: Retailers, urban consumers, bulk purchasers  
- **Commission Agents**: Market intermediaries facilitating transactions

### Primary Features
- Role-based authentication and user profiles
- Real-time price discovery with market trends
- AI-powered negotiation assistance
- Multilingual chat and voice communication
- Transaction management and reputation system
- Regional customization for local market practices
- Offline-capable PWA functionality

### Current Status
The application has a complete React frontend with UI components and navigation, but core backend features like translation engine, real-time data feeds, and AI services are still in development.
